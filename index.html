Patents

US 10,574,558 — Issued Feb 2020

Limiting Alarms in an Asset Monitoring System

A smart alarm suppression system that prevents alert fatigue in infrastructure monitoring. Uses one-time trigger conditions to suppress notifications until an alarm condition has been validated — reducing noise while keeping critical alerts front and center.

US 08849351.5 — Issued Nov 2007

BGP Route Server for Multi-Provider Networks

A route server architecture that connects users to multiple ISPs, maintaining multiple routing table views and BGP instances to optimize path selection. Think of it as a smart traffic director for internet routing — matching users to the best available path across competing providers.
